# GeoConvert
============
converting between geojson and gis type data(kml, gpx)

#Usage
------
kml to geojson
```javascript
GeoConvert.kml2Geojson(kml);
```

geojson to kml
```javascript
GeoConvert.geojson2Kml(geojson);
```

gpx to geojson
```javascript
GeoConvert.gpx2Geojson(kml);
```

geojson to gpx
```javascript
GeoConvert.geojson2Gpx(geojson);
```
